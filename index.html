JLCap

JLCap is a professional captcha generator for web apps or sites. You can easily develop without needing any type of framework, just keep the development with clean and clear code.

Features

  • Use normally in HTML img tags.
  • Cross-browser compatible (Chrome, Firefox, IE8+, Opera, Safari).
  • Select multiple images as background.
  • Great performance.

JLClass works in PHP version 5.2.4 or higher.
